# gocpp
Simple example of cmake project for building and static linking the go library to c/cpp application.

## Build steps

Clone project:
```
$ git clone https://github.com/WinPooh32/gocpp.git
$ cd gocpp
```

Create folder for generated makefiles:
```
$ mkdir build
$ cd build
```

Run cmake:
```
$ cmake ..
```

Build:
```
$ make
```

Run app:
```
$ ./app
```
